# todo-list-project
# Client Side Todo List Application

## Project Objective
The objective of this project is to create a simple Todo List web application that allows users to manage their daily tasks.

## Project Description
This application allows users to:
- Add tasks
- Mark tasks as completed


- Delete tasks

The application runs completely in the browser and uses JavaScript to manage tasks dynamically.


## Technologies Used
HTML5
CSS3
JavaScript

## How to Run the Project
1. Download the project files
2. Open index.html in any browser
3. Start adding tasks

## Features
Add Task
Mark Task Complete
Delete Task
Simple and user-friendly interface
